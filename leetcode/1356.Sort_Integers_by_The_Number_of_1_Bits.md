[根据数字二进制下 1 的数目排序](https://leetcode-cn.com/problems/sort-integers-by-the-number-of-1-bits/)

### 解题思路
1、先对数组进行从小到大排序
2、将数组中的元素对应的二进制进行分组
3、从分组后的重新构造成数组

### 代码

```golang
func sortByBits(arr []int) []int {
    m := make(map[int][]int)
    maxV := 0
    sort.Ints(arr)
    for i := 0;i < len(arr);i++ {
        num := 0
        for t := arr[i];t != 0;t /= 2 {
            num += t & 1
        }
        if num > maxV {
            maxV = num
        }
        m[num] = append(m[num], arr[i])
    }
    ans := make([]int, 0, len(arr))
    for i := 0;i <= maxV;i++ {
        ans = append(ans, m[i]...)
    }
    return ans
}
```
